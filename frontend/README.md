# Frontend — coming in Phase 4

The Next.js 14 (App Router) + TypeScript + Tailwind + Recharts + Leaflet
frontend is scaffolded in **Phase 4**.

It will faithfully recreate the Claude design (dark "command-center" UI):
- `/dashboard` — full-screen draggable Leaflet satellite map with floating panels
- `/routes` — 3-route comparison + toll plaza breakdown
- `/alerts` — live alert feed + history + subscribe widget
- `/emergency` — emergency routing planner with signal pre-emption
- Floating AI chatbot mounted in `app/layout.tsx`
